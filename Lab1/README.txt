Lab1 

How to run: Press the play button and flip the on/off switches

-----------
DESCRIPTION

In this lab, the user will turn the on/off switches up (on) or down (off) in order light up the LEDs.
Each switch represents a 1 or 0 in binary. For example 1111, 1010, 0000.


-----------
FILES

-
Lab1.lgi

This files contains the MultiMedia Logic project


-----------
INSTRUCTIONS

This program is intended to be run using the MultiMedia Logic software. 
Once loaded, press the play button towards the top of the program and flip
any switch up or down in order to light up an LED.
